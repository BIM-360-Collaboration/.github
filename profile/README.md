## 🚀 Introduction
BIM 360 for Windows is Autodesk's cloud construction management platform, created to unite the people, models, and documents behind a building project. It answers the everyday question of what is BIM 360 by acting as the shared workspace where drawings are controlled, designs are reviewed, models are coordinated, and field issues are tracked. Since BIM 360 is cloud-based and paired with a Windows desktop companion, distributed teams collaborate on the same information whether they are at a desk or on a job site. The platform is now delivered as part of Autodesk Construction Cloud, broadening its reach across the project lifecycle.

## ✨ Highlights

### Documents and design
- **Document management** that keeps every drawing, plan, and specification versioned, organized, and access-controlled.
- **Design collaboration** that lets disciplines review, mark up, and align on the current design intent together.

### Coordination and field
- **Model coordination** that aggregates trade models, runs automatic clash detection, and clears conflicts early.
- **Field and quality management** that captures issues, checklists, and inspections on site and syncs them back to the office.
- **Autodesk integration** that links Revit and the wider Autodesk ecosystem so data flows through your workflow.

## 💻 System Specs
**Minimum**
- OS: Windows 10 64-bit
- Processor: A modern 64-bit multi-core processor
- Memory: 8 GB RAM
- Storage: Space for the desktop companion and cached project files

**Recommended**
- OS: Windows 11 64-bit
- Processor: A recent high-performance 64-bit processor
- Memory: 16 GB RAM or more
- Storage: An SSD with plenty of free space for synced data

## 📥 How to Install
1. Click the **Download** button above to get the latest BIM 360 desktop companion for Windows.
2. Run the downloaded installer and follow the on-screen setup steps.
3. Launch BIM 360 and sign in or create an Autodesk account.

## 🎯 Getting Up and Running
1. Complete your BIM 360 login with your Autodesk credentials to open the project workspace.
2. Select a project and explore the document management area to reach drawings, sheets, and models.
3. Sync folders with the desktop connector so your Windows files match the cloud.
4. Invite collaborators, set permissions, and start coordinating models, reviewing designs, or logging field issues.

## ❓ Common Questions
**Is BIM 360 free?**
No. BIM 360 is a paid subscription platform from Autodesk. The BIM 360 license you hold reflects your plan and is now offered inside Autodesk Construction Cloud.

**Which versions of Windows are supported?**
BIM 360 supports current 64-bit versions of Windows, including Windows 10 and Windows 11. The main interface runs in a modern browser, and the desktop companion installs as a standard Windows application.

**Is my data secure?**
Yes. Autodesk keeps your project data in the cloud with detailed permission controls, so administrators decide exactly who can open or edit each folder, model, and issue.
